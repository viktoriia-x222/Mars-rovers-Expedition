# Mars-Rovers-Expedition

Start:
python main.py

Bewegung:
W A S D - Tasten
Beschreibung:
Der Rover bewegt sich durch eine zweidimensionale Welt mithilfe von x- und y-Koordinaten.
Er verbraucht Energie und reagiert auf zufällige Ereignisse.
Ziel:
Es ist das Ziel, die Zielposition zu erreichen und dabei die Energie über 0 zu halten.
